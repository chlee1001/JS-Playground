## drumkit

- 드럼 만들기 구현
- 키보드 입력으로 오디오 재생
- 클릭으로 오디오 재생

### 유사 배열 객체

- 배열이 아닌데 배열인척 하는 객체
- `querySelectorAll`로 dom을 선택하는 경우 Nodelist가 반환되는데, 이게 그 유사배열객체이다.
- 유사배열객체는 `Array.from` 을 사용해 배열로 만들 수 있다.
